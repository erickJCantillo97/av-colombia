# CalendarSelector

Este proyecto utiliza [`@vuepic/vue-datepicker`](https://vue3datepicker.com/) para el selector de fechas tipo calendario, con soporte para fechas exactas y rangos flexibles (± días) como en la interfaz de Airbnb.

## Instalación

Ejecuta en la raíz del proyecto:

```
npm install @vuepic/vue-datepicker
```

o si usas yarn:

```
yarn add @vuepic/vue-datepicker
```

Luego puedes usar el componente `CalendarSelector.vue` en cualquier parte de tu app Vue 3.

---

**Recuerda instalar la dependencia antes de ejecutar el proyecto.**
